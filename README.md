# PatentLingo

## Project Overview
Multilingual Legal Translation Assistant for Indian Patent Specifications (English to Kannada)

## Features
- True semantic translation (not transliteration)
- Fine-tuned on OPUS and Tatoeba corpora
- Prompt engineering for legal/technical precision
- Preserves document structure
- Consistent patent terminology mapping

## Datasets Used
- OPUS Corpus (EN-Kannada) - Primary training data
- Tatoeba Parallel Sentences - Validation and testing
- Patent Terminology Glossary - Domain-specific terms

## Tech Stack
- Model: IndicTrans2 (AI4Bharat)
- Framework: PyTorch, Transformers
- Training: Google Colab (T4 GPU)
- Deployment: Hugging Face Spaces
